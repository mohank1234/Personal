# Girlfriend Happiness Dashboard 💘

A funny, flirty, apology-style single-page website.

## Local preview

```bash
cd /workspace/Personal
python3 -m http.server 8000
```

Open <http://localhost:8000>.

## Free deployment on GitHub Pages

This repo includes an automatic GitHub Pages workflow:

- Workflow file: `.github/workflows/deploy-pages.yml`
- Trigger: push to `main` or `work`

### One-time setup in GitHub

1. Push this repository to GitHub.
2. Open **Settings → Pages**.
3. In **Build and deployment**, choose **GitHub Actions** as source.
4. Push to `main` (or `work`) to trigger deployment.

### Ready-to-use URL format

Once deployed, your site URL will be:

`https://<your-github-username>.github.io/<your-repo-name>/`

If your repo name is `Personal`, it will look like:

`https://<your-github-username>.github.io/Personal/`
